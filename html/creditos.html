Créditos

De onde veio isso?

Esse site é apenas um marcador para um tutorial do excelente gerenciador de conteúdo ModX. Eu precisava de uma base em HTML para converter em modelos no ModX então aproveitei para testar o novo Zurb Foundation 3 e montar um protótipo.

Abaixo alguns créditos das ferramentas e inspirações para o desenvolvimento:

O Tutorial

O tutorial sobre ModX Revolution está em produção e os videos estão no Youtube. Aproveite para se inscrever e também visitar o blog.

ModX

Um ótimo CMS open source. Ótimo para designers, você acrescentar porções do ModX em seu HTML sem se preocupar em seguir e modificar padrões de layout estabelecidos.

Foundation 3

Zurb Foundation é um framework responsivo para desenvolvimento em HTML, CSS e JS. Na versão 3 há opção de baixar o framework em SASS.
